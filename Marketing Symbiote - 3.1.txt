### **Coalition Lifecycle (The 6 Phases)**

```python
class Coalition:
    def __init__(self, trigger, urgency, agents):
        self.trigger = trigger
        self.urgency = urgency
        self.agents = agents
        self.sync_cycle = "15min" if urgency == "HIGH" else "60min"
        
    def execute(self):
        # Phase 1: Detection
        trigger_condition = AIT.detect_anomaly()
        
        # Phase 2: Formation
        coalition = ADA.assemble_team(trigger_condition)
        
        # Phase 3: Execution (Parallel workflows)
        for agent in coalition.agents:
            agent.execute_parallel(sync_cycle=self.sync_cycle)
        
        # Phase 4: Approval Gates
        if critical_checkpoint:
            await_human_approval(you)
        
        # Phase 5: Disbanding
        coalition.dissolve_upon_resolution()
        
        # Phase 6: Learning
        RAG.index_learnings(coalition.post_mortem())
```
